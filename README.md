## ⏳ Foydalanish  
1. Ushbu repozitoriyani yuklab oling yoki klonlang:  
   ```bash  
   git clone https://github.com/whdev36/python-kodlar.git  
   ```  
2. Python fayllarni ishga tushirish uchun terminalda quyidagicha yozing:  
   ```bash  
   python filename.py  
   ```  
   yoki  
   ```bash  
   python3 filename.py  
   ```
